oppia.core.tests.reader_view_load_test module¶

Performance test for the Oppia reader view.

Before running this script, exploration 0 should be loaded in the target server.

Run this script from the Oppia root directory:

python core/tests/reader_view_load_test.py –thread_count=5 –start_uid=1 https://my-oppia-instance.appspot.com
