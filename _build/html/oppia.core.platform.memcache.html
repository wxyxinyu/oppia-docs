oppia.core.platform.memcache.gae_memcache_services.get_multi(keys)[source]¶

Looks up a list of keys in memcache.

Parameters:keys (-) – a list of keys (strings) to look up.
Returns:A dict of key-value pairs for the keys/values that were present in memcache.
oppia.core.platform.memcache.gae_memcache_services.set_multi(key_value_mapping)[source]¶

Sets multiple keys’ values at once.

Parameters:key_value_mapping (-) – a dict of {key: value} pairs. The key is a string and the value is anything that is serializable using the Python pickle module. The combined size of each key and value must be < 1 MB. The total size of key_value_mapping should be at most 32 MB.
Returns:A list of the keys whose values were NOT set.
