class oppia.extensions.objects.models.objects.BaseObject[source]¶

Bases: object

Base object class.

This is the superclass for typed object specifications in Oppia, such as SanitizedUrl and CoordTwoDim.

Typed objects are initialized from a raw Python object which is expected to be derived from a JSON object. They are validated and normalized to basic Python objects (primitive types combined via lists and dicts; no sets or tuples).
