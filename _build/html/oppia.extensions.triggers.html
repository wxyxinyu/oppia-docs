oppia.extensions.triggers.trigger_classes module¶

Base class for defining triggers.

Although this module is in extensions/, it is not provided as an extension framework for third-party developers. This is because reacting to triggers involves changes to core code.

class oppia.extensions.triggers.trigger_classes.BaseTrigger[source]¶

Bases: object

Base trigger definition class.

This class is not meant to be user-editable. The only methods on it should be get()-type methods.

class oppia.extensions.triggers.trigger_classes.NthResubmission[source]¶

Bases: oppia.extensions.triggers.trigger_classes.BaseTrigger

This trigger is invoked when an answer is submitted to the same state for the nth time in succession, and the destination that would result due to normal evaluation would cause a further loop-around to the same state.
